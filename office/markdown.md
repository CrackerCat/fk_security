# markdown
## syntax
* \[链接文字\](链接地址 "标题") 插入链接
* !\[alt text\](图片相对路径/绝对路径 "title") 插入图片
* --- 分割线
* **文字** 加粗
* &emsp; 全角空格
* 四个空格或者一个制表符 ： 列表项目的段落格式  
* <sup>num</sup> : 上标
* <sub>num</sub> : 下标
